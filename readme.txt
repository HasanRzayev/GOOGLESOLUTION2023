1. Our project is developed using Visual Studio application.
2. We used C# framework ASP.NET with MVC architectural pattern.
3. For Database we used Microsoft SQL Server.
4. To run this application firstly change the connection string in the appsettings.json file to your particular database connection string.
5. Then open Package Manager Console from Tools.
6. Add migration using "add-migration init" command.
7. Then update database using "update-database" command.
8. Finally run project.
